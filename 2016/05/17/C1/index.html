Before interview, practice both using and implementing hash tables
C++11 STL中 unordered_map 实现了 hash_table，下面是一个例子:

实现一个hash_table, 能实现简单的插入，删除功能

ArrayList(动态数组)

  • 可以动态调整大小的数组。C++中用vector来实现。预先开辟一个固定大小的空间，如果空间用完了，就再开辟一个新的两倍大小空间，然后把原数组复制过去。平均时间复杂度为O(1).

String

C++提供的String类功能十分强大，例如比较，连接，搜索，替换，获得子串等等。并且重载了许多操作符： 包括 +, +=, <, =, , [], <<, >>等，这些操作符对字符串操作非常方便。有了操作符，assign(), append(), compare(), at()等函数，除非是特殊需求，一般是用不上的。

题目：

1.1 Implement an algorithm to determine if a string has all unique characters. What if you can not use additional data structure?

1.2 Write code to reverse a C-style String(C-String means that “abcd” is represented as five characters, including the null character)

1.3 Design an alogrithmand write code to remove the duplicate characters in a string without using any additional buffer. NOTE:One or two additional variables are fine. An extra copy of the array is not. FOLLOW UP: Write the test cases for this method

1.4 Write a method to decide if two strings are anagrams or not

1.5 Write a method to replace all spaces in a string with ‘%20’
